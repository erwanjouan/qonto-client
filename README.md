# qonto-client

- AutoConfiguration to retrieve transaction from Qonto bank

## Usage

- ```mvn clean install``` and add maven dependency to projet using client

```
<dependency>
<groupId>com.theatomicity.qonto</groupId>
<artifactId>client</artifactId>
<version>0.0.1-SNAPSHOT</version>
</dependency>

```

## Environment variables (API Key)

- QONTO_ID
- QONTO_API_KEY

## Api Documentation
